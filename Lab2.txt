Вариант 11
Задание 1
Описать процедуру Sort(A, N), выполняющую сортировку по возрастанию вещественного массива A размера N.
Массив A является входным и выходным параметром. С помощью этой процедуры отсортировать массивы A, B, C размера NA, NB, NC соответственно.
Задание 2
Описать функцию WordK(S, K) строкового типа, возвращающую K-е слово строки S (словом считается набор символов, не содержащий
пробелов и ограниченный пробелами или началом/концом строки). Если количество слов в строке меньше K, то функция возвращает
пустую строку. Используя эту функцию, выделить из данной строки S слова с данными номерами K1, K2, K3.
Задание 3
Описать рекурсивную функцию MaxInt(A, N) целого типа, которая находит максимальный элемент целочисленного массива A размера N (1 ≤ N ≤ 10),
не используя оператор цикла. С помощью этой функции найти максимальные элементы массивов A, B, C размера NA, NB, NC соответственно.
